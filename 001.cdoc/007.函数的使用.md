# 函数的使用

## 1. 函数值传递
```c
#include <stdio.h>
// 函数值传递
int sum(int a, int b)
{
    return a + b;
}

int main(int argc, char const *argv[])
{
    // 值传递的函数
    int num_01 = 10;
    int num_02 = 20;
    int res = sum(num_01, num_02);
    printf("%d + %d = %d", num_01, num_02, res);
    printf("\n");
}
```


## 2. 引用传递

```c
#include <stdio.h>
// 引用传递
void swap(int *a, int *b)
{
    int t = *a;
    *a = *b;
    *b = t;
}

int main(int argc, char const *argv[])
{
    // 引用传递
    int num_11 = 10;
    int num_12 = 20;
    printf("num_11: %d; num_12: %d", num_11, num_12);
    printf("\n");
    swap(&num_11, &num_12);
    printf("num_11: %d; num_12: %d", num_11, num_12);
    printf("\n");
    return 0;
}
```

## 3. 返回 static 变量的指针

```c
#include <stdio.h>

// 返回 static 修饰的指针
int *create_arr()
{
    static int arr[5];
    arr[0] = 11;
    arr[1] = 12;
    arr[2] = 13;
    arr[4] = 15;
    return arr;
}

int main(int argc, char const *argv[])
{
    int *ptr_51;
    ptr_51 = create_arr();
    for (int i = 0; i < 5; i++)
    {
        printf("第 %d 个元素, 元素的值为: %d\n", i, *ptr_51);
        ptr_51++;
    }
}
```
